# HR Management Portal

## Overview

HR Management Portal is a web application developed using Django and MySQL. It helps organizations manage employee information, attendance records, payroll details, and dashboard statistics.

## Technologies Used

- Python
- Django
- MySQL
- HTML
- CSS
- Bootstrap

## Features

- Secure Login and Logout
- Employee Management (CRUD Operations)
- Attendance Management
- Payroll Management
- Dashboard Statistics

## Project Structure

- Accounts Module
- Employee Module
- Attendance Module
- Payroll Module
- Dashboard Module
